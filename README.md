# QazaqCompetition MVP

MVP веб-проект для регистрации на соревнования в Казахстане.

## Стек
- Frontend: HTML, CSS, Vanilla JS
- Backend: Node.js + Express
- DB: SQLite
- Auth: JWT + роли

## Функции MVP
- Регистрация и вход
- Роли: участник, судья, организатор, зритель, админ
- Каталог соревнований + фильтры
- Регистрация участника на соревнование
- Зал славы (награды, опыт, история матчей)
- Live и видео-архив
- Бонусы и история начислений/списаний
- Профиль пользователя (автосоздание при регистрации + редактирование)
- Судейская панель (оценка + QR-проверка)
- Панель организатора (создание соревнований, услуги, AI-жеребьевка)
- AI-функции: советы, шансы, слабые места, жеребьевка
- Ролевая навигация (показываются только нужные разделы для выбранной роли)

## Структура
- `server.js` - Express API и раздача статики
- `src/db.js` - схема БД и seed-данные
- `src/auth.js` - JWT middleware
- `script.js` - клиентская логика
- `style.css` - единый стиль для всех страниц
- `*.html` - страницы MVP

## Команды запуска
1. Установи Node.js 18+ (если еще не установлен)
2. Установи зависимости:
```bash
npm install
```
3. Запусти проект:
```bash
npm start
```
4. Открой:
- `http://localhost:3000`
